## 📝 Sobre o projeto
Esta é uma API RESTful desenvolvida em Flask para o gerenciamento de granjas, permitindo o controle de produção, recursos e logística.

## 🛠️ Tecnologias utilizadas
- **Python 3.13**
- **Flask** (Framework Web)
- **Flask-RESTful** (Extensão para APIs REST)
- **SQLAlchemy** (ORM para banco de dados)
- **Marshmallow** (Serialização e validação de dados)
- **Pytest** (Testes automatizados)
